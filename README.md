# nova-desktop

Pika desktop app — Electron wrapper for [pika.me](https://pika.me)

## Install

Download the latest `.dmg` from [Releases](../../releases).

Open the `.dmg`, drag **Pika** to your Applications folder, and launch.

## Features

- Native macOS window with traffic light controls
- Dark mode
- Opens external links in your default browser
- Keyboard shortcuts (⌘R reload, ⌘+/- zoom, etc.)

## Dev

```bash
npm install
npm start        # dev mode
npm run build    # build .dmg
```

## Stack

- Electron 34
- electron-builder
